# @guarddiff/cli

Command-line interface for GuardDiff.

Most users should install the public front-door package:

```bash
npm install -g guarddiff
guarddiff staged
```

Configured external rule packs are disabled unless you pass `--allow-rule-packs` for a trusted repository.

Use `npm install -g @guarddiff/cli` only when you specifically want the scoped package.
