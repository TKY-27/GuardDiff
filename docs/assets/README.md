# Assets

This directory is for public documentation assets only.

- Do not commit raw terminal recordings, screenshots, or videos that show usernames, hostnames, home directories, emails, private repository names, shell history, or real secrets.
- Prefer short, deterministic CLI output examples over recordings when a demo might expose local context.
- Re-scan `.cast`, image, video, and log assets before every public release.
- Keep real credentials out of fixtures. Use obvious placeholders and confirm reporters mask matched values.
- SVG demo images should use synthetic paths, synthetic findings, and masked token examples only.
